Heroes
partials/headers/header-basic: un tentativo di poter gestire tutte le tipologie di header scrivendo meno codice possibile. Il partial si aspetta un oggetto page che contenga smepre almeno title e description e opzionalmente summary Il partial accetta alcuni parametru opzionali:
  • share: true per mostrare il meno di condivisione social,
  • classes: "bla bla" per aggiungere classi aggiuntive all'header, cambaire colore di sfondo, del testo ecc
  • subs: [collection] per mostrare il menu degli approfondimenti (subpages)
È necessario poi passare un blocco che contenga content_for [:area] o che almeno finisca con un do lasciando il blocco vuoto. Le aree dispnibili sono: :aside, :col_one, :col_two. Ogni blocco dentro content_for può avere codice a piacimento
